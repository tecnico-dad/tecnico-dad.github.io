Avaliação de projetos

Durante esta semana os professores dos laboratórios estão a avaliar as entregas intermédias dos projetos.
Se tiverem alguma informação relevante a acrescentar ao que foi entregue, devem contactar com o professor do vosso turno por email ou no canal de texto do vosso grupo no Discord.

Entretanto, devem aproveitar esta semana para realizar o laboratório ZooKeeper para aprender como usar este servidor de nomes para registar e pesquisar serviços gRPC, tal como vai ser feito na segunda parte do projeto.
Os professores estarão no Discord durante os horários das aulas para esclarecimento de dúvidas.
