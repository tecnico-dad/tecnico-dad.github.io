Apoio ao Projeto

O objetivo desde laboratório é a implementação das operações do serviço Sentry, feed e ctrl_ping, componente do projeto SAYF.

Relembre que, caso não o tenha feito, deve se inscrever no Piazza para esclarecimento de dúvidas fora dos laboratórios.

Tópicos a revisitar / qualidade de código

  • Testes de Integração
  • Tratamento de exceções
  • Concorrência e Sincronização

Sentry

O servidor sentry recebe observações feitas pelo rádio e depois envia um conjunto de observações acumuladas para armazenamento.

  • Estude a especificação do servidor sentry presente no enunciado do projeto.
  • Obtenha o esqueleto do projeto do repositório git atribuído a cada grupo.
    • O repositório do grupo CXX está em: https://github.com/tecnico-distsys/CXX-SAYF, onde CXX deve ser substituído por T e os dois dígitos do número do grupo.

sentry-contract

Comece por detalhar no ficheiro Sentry.proto as operações RPC.

  • Crie as operações feed e ctrl_ping.
  • Relembrar: Tutorial de gRPC para Java.

sentry-server

Implemente no servidor as operações remotas que definiu no ficheiro Sentry.proto.

  • Relembrar: API de gRPC para Java.
  • Implemente a lógica do servidor para as operações feed e ctrl_ping.

sentry-client

Implemente no cliente uma classe que instancia o stub do serviço, e adicione métodos para invocar as operações remotas.

  • Implemente os métodos para as operações feed e ctrl_ping.

Implemente testes de integração, de modo a verificar o correto funcionamento do servidor sentry.

  • Observe a classe SentryIT. Observe as anotações: @BeforeClass, @AfterClass, @Before, @After, @Test.
  • Relembrar: Testes de Integração
  • Crie testes para o serviço RPC. Estes testes devem incluir as operações feed e ctrl_ping.

ZooKeeper Naming

Os vários componentes do projeto devem descobrir-se uns aos outros através do servidor de nomes ZooKeeper.

  • Relembrar aula de zookeeper.
  • Obter biblioteca zk-naming [atualizado] .
  • Instalar a biblioteca no repositório local maven.

No final desta aula deve conseguir correr os testes de integração, verificando que consegue fornecer dados para o servidor Sentry com o comando feed e obter os mesmos com o comando ctrl_ping.
